Bisulfite cloning grid viewer

Upload a .txt file: first sequence is the reference (unconverted), the rest are Sanger reads from individual clones, all aligned to the same length. Works for DNA (5mC, typically CpG) or RNA (m5C, any sequence context) bisulfite data, choose which C-context(s) to score below. Scored cytosines are white (converted / unmodified), black (unconverted / modified), or orange (mismatch, gap, or unscored change).

1. Load data

Accepted format: FASTA (>name header followed by sequence) is preferred. Plain text with one sequence per line (first line = reference) also works. Non-letter characters are stripped automatically.

2. Sites to score

Select which base immediately 3' of a reference C should count as a scored site (e.g. CG for classic CpG methylation, or CA/CC/CT for non-CpG contexts common in RNA m5C). Unselected C positions are used only as a bisulfite-conversion efficiency control.
Show every position in the chosen window, not just C's. The reference sequence's base identity is shown in a row above the grid. Inside the grid: white = matches reference; black = reference C read as C (unconverted); grey = reference C read as T (converted); orange = any other mismatch.
